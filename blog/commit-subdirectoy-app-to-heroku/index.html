Commit subdirectoy application to Heroku

— October 27, 2013

Directory structure:

.
├── LICENSE
├── README.md
├── express
│   ├── Procfile
│   ├── README.md
│   ├── app.js
│   ├── models
│   │   └── user.js
│   ├── ormConfig.js
│   ├── package.json
│   └── seeds.js
└── railsapi
    ├── Gemfile
    ├── Gemfile.lock
    ├── Procfile
...

If you want to commit any of the projects express or railsapi to Heroku you can use this command:

$ git subtree push --prefix railsapi heroku master
